# EMG-Based Motor Training System

This project combines a serious game and a software component for motor training using EMG (electromyography) data. It is designed to support rehabilitation and neuromuscular therapy, particularly for individuals with movement disorders.

## 🧩 Components

- **C# (Unity)**: Interactive serious game with real-time feedback based on EMG signals.
- **Python**: EMG signal processing, feature extraction, and classification.

## 🎮 Game Features

- Adaptive difficulty based on muscle activity
- Visual and auditory feedback for engagement
- Designed for therapeutic motor exercises

## 📊 EMG Analysis

- Signal preprocessing and filtering
- Feature extraction (e.g., RMS, MAV, ZC)
- Classification of muscle activation patterns

## 🚀 Getting Started

1. Clone the repository
2. Run the Unity project for the game interface
3. Use Python scripts to process EMG data and feed results into the game

## 📁 Folder Structure
/UnityGame/ # C# Unity project for the serious game 
/PythonEMG/ # Python scripts for EMG data processing
